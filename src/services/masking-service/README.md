# Masking Service

Dịch vụ che giấu dữ liệu nhạy cảm - áp dụng K-anonymity và tokenization.

## Chức năng chính

- Che giấu dữ liệu theo K-anonymity
- Tokenization dữ liệu nhạy cảm
- Phi cá nhân hóa dữ liệu trước khi chia sẻ
- Quản lý chính sách masking

## Hạ tầng phụ thuộc

- PostgreSQL (masking policies, metadata)
