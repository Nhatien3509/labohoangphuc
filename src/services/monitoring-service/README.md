# Monitoring Service

Dịch vụ giám sát hệ thống - theo dõi trạng thái, cảnh báo, và dashboard thời gian thực.

## Chức năng chính

- Dashboard giám sát thời gian thực
- Theo dõi tính khả dụng dịch vụ
- Quản lý cảnh báo (alert rules)
- Nhật ký hệ thống và tích hợp

## Hạ tầng phụ thuộc

- PostgreSQL (alert rules, dashboard config)
- SigNoz (OpenTelemetry - logs/metrics/traces)
